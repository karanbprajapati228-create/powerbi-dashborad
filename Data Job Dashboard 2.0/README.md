# 📊 Data Jobs Dashboard 2.0

## Overview
The **Data Jobs Dashboard 2.0** provides an interactive view of the data job market, highlighting job counts, required skills, and salary insights. It is designed to help professionals, recruiters, and learners understand industry trends and make informed career decisions.

---

## 🔑 Key Metrics
- **Total Job Count:** 479K  
- **Average Skills per Job:** 4.8  
- **Median Hourly Salary:** $48  
- **Median Yearly Salary:** $113K  

---

## 💼 Top Paying Jobs in Data
The dashboard showcases the highest-paying roles in the data industry, with median salary ranges up to **$200K/year**.  
Highlighted roles include:
- Senior Data Scientist  
- Machine Learning Engineer  
- Senior Data Engineer  
- Software Engineer  
- Cloud Engineer  
- Data Scientist  
- Senior Data Analyst  
- Business Analyst  
- Data Analyst  

---

## 🛠️ Top Skills in Demand
The dashboard tracks the most requested skills across data jobs.  
Key skills include:
- **Programming & Analytics:** Python, R, SQL, Java  
- **Cloud & Big Data:** AWS, Azure, Spark, Hadoop, Databricks, Snowflake  
- **Visualization & BI:** Tableau, Power BI, Excel  

---

## 📈 Dashboard Features
- **Interactive Filters:** Select job titles or countries to refine insights.  
- **Salary Distribution:** Compare median hourly and yearly salaries across roles.  
- **Skill Frequency:** View job counts and percentages for top skills.  
- **Dynamic Visuals:** Histograms, bar charts, and pie charts for easy interpretation.  

---

## 🎯 Use Cases
- **Job Seekers:** Identify high-paying roles and in-demand skills.  
- **Recruiters:** Track market trends and skill requirements.  
- **Learners & Students:** Focus on skills that maximize employability.  
- **Organizations:** Benchmark salaries and skill expectations.  

---

## 📌 How to Use
1. Open the dashboard in Power BI.  
2. Use slicers to filter by **Job Title** or **Country**.  
3. Explore salary insights and skill distributions.  
4. Apply findings to career planning, recruitment, or training programs.
